算数运算符

+: 加法 、取正

int a = 1，b = 2;
String str = "hello";
System.out.println(a + b + str);//3hello
System.out.println(str + a + b);//hello12
System.out.println(str + (a + b));//hello3
System.out.println(a + str + b);//1hello2

从左向右执行：

  • 符号两边有任意一边是字符串 做的是字符串拼接
  • 符号两边都是数字 做的是加法
int x = 1;
int y = 3;
char c = '0';//48
String s = "0";
System.out.println(x + y + c);//52
System.out.println(s + y + c);//030
System.out.println(y + c + s);//510

System.out.println("----------------------------------");
System.out.println("*	*");//*	*
System.out.println('*' + '\t' + '*');//数字
System.out.println('*' + "\t" + '*');//*	*
System.out.println('*' + '\t' + "*");//数字*
System.out.println('*' + ('\t' + "*"));//*	*

- : 减法 、 取负

* : 乘法

\ : 除法

​ System.out.println(10/3*3);//9

%：取余

  • 任意数字和n取余结果是[0，n-1]

  • 取余结果符号和被除数符号一致

int a = 10;
int b = 3;
System.out.println(a  + " % " + b  + " = " + (a % b));
System.out.println(-a + " % " + b  + " = " + (-a % b));
System.out.println(a  + " % " + -b + " = " + (a % -b));
System.out.println(-a + " % " + -b + " = " + (-a % -b));

/*
	随意给出一个整数，打印显示它的个位数，十位数，百
	位数的值，例如:数字345的情况如下：
	个位数：5 十位数：4 百位数：3
	*/
int number = 345;
System.out.println("个位数:" + number%10);
System.out.println("十位数:" + number/10%10);
System.out.println("百位数:" + number/100);//number/10/10

前++ 、 后++、前–、后–

++

  • 同:运算结果相同 都是减1
  • 异:前++：先加1 再赋值 后++：先赋值 再加1

–

  • 同:运算结果相同 都是减1
  • 异:前–：先减1 再赋值 后–：先赋值 再减
  • ++ – 不改变自身数据类型 所以 byte b = 3 ;b++;不报错，区别于数据类型中的常数赋值
byte b = 3;
//b = b + 1;//报错
b++;//++b  === (byte)(b + 1)

赋值运算符

基本运算符： =

扩展运算符： += -= *= /= %=

//给一个数加1 有几种方式
i = i + 1;//方式1
i++;//++i; 方式2
i += 1;//方式3
  • 不改变自身数据类型 byte b = 5; b += 1; 不报错
byte b = 5;
//b = b + 1; //报错
b += 1;	//不报错 b = (byte)(b + 1)
int ii = 3;
ii *= 0.1;//不报错 ii = (int)(ii * 0.1)

关系运算符

逻辑运算符

操作数是boolean 运算结果也是boolean类型

逻辑与、短路与

  • 有一假结果是假
  • 同:运算结果相同
  • 当运算符左侧是false,&右侧继续运算 && 发生短路 右侧不运算
//& 和 &&短路与的区别
int age = 10;
char sex = '男';
//boolean result = sex == '女' & age++ == 10;	//age = 11
boolean result = sex == '女' && age++ == 10;		//age = 10

逻辑或、短路或

  • 有一真结果为真
  • 运算结果相同
  • 当运算符左侧是true,|右侧继续运算 || 发生短路 右侧不运算
//| 和 || 的区别
int age = 10;
char sex = '男';
//boolean result = sex == '男' | age++ == 10;	//age = 11
boolean result = sex == '男' || age++ == 10;		//age = 10

逻辑非

  • ！取反

逻辑异或

  • ^ :相同为假 不同为真

位运算符

操作的是数值,得到数值的补码然后进行位运算

移位运算(一元运算符)

左移 <<

每移动一位相当于* 2 空位补0

右移 >>

每移动一位相当于/ 2 最高位是0就补0 是1就补1

无符号右移 >>>

每移动一位相当于/ 2 最高位补0

逻辑运算(二元运算符)

每一位上如果是1 看做true 如果是0 看做false

int x = 12;
int y = 5;

逻辑与 &

int result = x & y;//4
//00000000 00000000 00000000 00001100 12
//00000000 00000000 00000000 00000101 5
//---------------------------------------
//00000000 00000000 00000000 00000100 4

逻辑或 |

result = x | y;//13
//00000000 00000000 00000000 00001100 12
//00000000 00000000 00000000 00000101 5
//---------------------------------------
//00000000 00000000 00000000 00001101 13

逻辑非 ~

result = ~x;//-13
//00000000 00000000 00000000 00001100 12
//--------------------------------------
//11111111 11111111 11111111 11110011 补码
//11111111 11111111 11111111 11110010 反码	补码减去1得到反码
//10000000 00000000 00000000 00001101 源码 	符号位不变其他位取反得到源码	- 13

逻辑异或 ^

result = x ^ y;//9
//00000000 00000000 00000000 00001100 12
//00000000 00000000 00000000 00000101 5
//---------------------------------------
//00000000 00000000 00000000 00001001 9
  • m = (m^^n)^^n

3种变量交换

临时变量

int tmp = a;
a = b;
b = tmp;

累加消除

b = a + b;
a = b - a;
b = b - a;
  1. 少定义一个变量
  2. 有局限，可能超过int类型

异或消除

b = a ^ b;
a = a ^ b;
b = a ^ b;
  1. 少定义一个变量
  2. 没有局限
  3. m = (m^^n)^^n

三元运算符

  • boolean表达式 ? 表达式1 : 表达式2

  • boolean表达式返回true 表达式1的结果作为整个表达式的结果

  • boolean表达式返回false 表达式2的结果作为整个表达式的结果

注意:

  1. 表达式1 和 表达式2 结果类型必须保持一致
  2. 表达式结果会发生自动类型转换，转换成范围大的数据类型
int age = 18;
String isAdult = age >= 18 ? "已成年" : 1.0;//报错

double r = true ? 1 : 1.0;	//不报错 发生自动类型转换 1.0
